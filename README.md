Overview：简介
本系统是一个药品供应链及处方执行的系统。主要包含药品管理、患者管理、药房管理、药房药品分配、处方创建和执行流程等

Installation：安装说明
1, JDK17+  Maven3.6+  Mysql8.0+(推荐Mysql9.3)  Git
2, 克隆项目： git clone https://github.com/jing747556248/pharma.git
3, 构建项目： mvn clean install
